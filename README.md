# Todo_Flask
Creating a Todo app with python flask and MongoDB. The frontend will be in React with Vite.js.

## How to start off
1. Clone the repo to your system.<br>
2. Ensure Python is installed in your system.<br>
3. Inside the main directory, run ```python3 -m venv venv``` to create a virtual environment<br>
4. Start the virtual environment(If on windows and running on cmd, run ```venv\Scripts\activate```<br>
5. Inside the backend directory, run ```pip install flask marshmallow flask-cors flask-pymongo```<br>
6. Inside the front directory, run ```npm init```, and ```npm install```<br>
7. Connect to the MongoDB server<br>
8. In a cmd terminal, go to the backend directory, and run ```flask run```<br>
9. In another cmd terminal, go to the frontend directory, and run ```npm run dev```<br>
10. In your preferred web browser, go to localhost:3000<br>
11. Your website is up and ready!
